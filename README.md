# Sentinel Protocol

Sentinel Protocol is an open-source AI safety ecosystem designed to function as an "immune system" for AI systems.

It enables continuous adversarial testing, evaluation, and governance of large language models.

## Core Components

- Red Set ProtoCell (RSP)
- ARES Dashboard
- AI Control Plane
